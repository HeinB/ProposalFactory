## Section 1: Oracle E-Business Suite Platform Overview

Oracle E-Business Suite (Oracle EBS) is Oracle Corporation's comprehensive, integrated enterprise resource planning platform, designed for organisations that require a robust, deeply configurable, and highly scalable ERP environment. Oracle EBS integrates financial management, human capital management, supply chain, procurement, project accounting, manufacturing, and customer relationship management within a unified, process-driven architecture.

Oracle EBS is available for deployment in three models:

- **On-premises** — Traditional hosted deployment on client infrastructure
- **Cloud-hosted on Oracle Cloud Infrastructure (OCI)** — EBS running on Oracle's own cloud, retaining on-premise architecture while gaining OCI security, availability, and disaster recovery benefits
- **Managed by APPSolve on OCI** — APPSolve manages the full EBS environment on OCI on behalf of the client

### 1.1 Core EBS Capability Areas

| Pillar | Description |
|---|---|
| **Financial Management** | General Ledger, Accounts Payable, Accounts Receivable, Fixed Assets, Cash Management, iExpenses, iReceivables, iSupplier, Subledger Accounting |

### 10.1 KPMG SADC Rollout — 6-Country Oracle EBS Deployment

APPSolve was contracted as the deployment partner for KPMG's Oracle EBS rollout across six SADC-region countries: **Botswana, Mauritius, Mozambique, Zambia, Zimbabwe, and Namibia**.

**Scope of deployment:**
- Oracle EBS Financial Management (GL, AP, AR, FA, Cash Management)
- Oracle EBS Human Capital Management (HRMS, Payroll)
- Oracle Projects / Billing Workbench / Oracle Time and Labour
- iExpenses, iProcurement
- Standard consolidation and inter-company accounting

**Implementation approach:**
- Initial deployment: Namibia (Time and Materials)
- Full SADC rollout followed a staged country-by-country plan
- "Train the Trainer" strategy deployed to empower KPMG local users
- Review of in-country processes to eliminate off-system (Excel) activities and drive adoption
- Transparent project governance structure aligned to KPMG's project sponsor team
- PRINCE2 project management methodology applied

**Methodology:** APPSolve's Oracle implementation approach combines PRINCE2 project management with Oracle's Implementation Lifecycle Model, creating a governance layer that protects implementation integrity. This methodology is applicable to any multi-country Oracle EBS or Oracle Cloud deployment.
