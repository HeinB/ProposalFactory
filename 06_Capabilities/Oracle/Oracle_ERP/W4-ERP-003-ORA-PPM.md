## Section 1: Statement of Capability

APPSolve implements and supports Oracle Project Portfolio Management (PPM) — Oracle's enterprise project and portfolio management platform, available on both Oracle EBS and Oracle Fusion Cloud. APPSolve has delivered Oracle PPM implementations in multi-country, multi-entity environments across professional services and financial services sectors.

## Section 2: Product Architecture

### 2.1 Oracle PPM on EBS and Fusion Cloud

Oracle PPM is available in two platforms — Oracle EBS (PA module) and Oracle Fusion Cloud (Project Management). APPSolve has delivered Oracle PPM on Oracle EBS (KPMG — multi-country). Oracle Fusion PPM is the next-generation platform.

```
Oracle PPM — Project Lifecycle
  ├── Project Initiation
  │   ├── Project templates
  │   ├── Project hierarchy (programme → project → task)
  │   └── Budget baseline
  ├── Project Execution
  │   ├── Time and expense capture (linked to Oracle HR or timesheets)
  │   ├── Progress tracking (task completion %)
  │   └── Issue and risk management
  ├── Project Accounting (integrated with Oracle Financials)
  │   ├── Cost collection (labour, materials, overhead)
  │   ├── Contract billing (fixed price, T&M, milestone)
  │   └── Revenue recognition (IFRS 15 compliant)
  ├── Resource Management
  │   ├── Resource demand planning
  │   ├── Utilisation and capacity analytics
  │   └── Cross-project resource allocation
  └── Portfolio Analytics
      ├── Portfolio health dashboards
      ├── Budget vs. actuals tracking
      └── Executive PMO reporting (OTBI)
```

### 2.2 Integration with Oracle ERP

Oracle PPM integrates natively with Oracle Financials:
- Costs flow from PPM to Oracle GL (General Ledger)
- Revenue recognised in Oracle Financials from project billing events
- Supplier invoices (AP) can be charged to projects
- Payroll costs from Oracle HCM or third-party can be allocated to projects
