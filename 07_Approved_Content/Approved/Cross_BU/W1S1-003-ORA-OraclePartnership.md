# APPSolve Oracle Partnership

APPSolve has been an Oracle Partner for over two decades. Our reputation and good standing in the Oracle community can be leveraged to ensure that your company receives the best possible support to address any issues.

## Partner Credentials

APPSolve holds **Oracle Level 1 Partner** status under Oracle's current partner model, with the following published expertise:

- **Oracle Fusion Cloud Financials** — implementation expertise certified
- **Oracle Fusion Cloud HCM Core** — implementation expertise certified
- **Oracle Integration** — implementation expertise certified
- **Oracle E-Business Suite Migration to OCI** — migration expertise certified
- **Oracle Cloud Infrastructure Migration** — migration expertise certified

APPSolve is also:

- **Oracle Value-Added Reseller** — authorised to sell and distribute Oracle licenses
- **Oracle Cloud Excellence Implementer** — certified
- **Oracle Enterprise Linux Specialist** — specialised competency

APPSolve's Oracle Level 1 Partner status entitles us to:

- Sell and distribute Oracle licenses
- Provide clients with access to real-time global Oracle support
- Implement solutions sustained by Oracle's global knowledge base
- Leverage Oracle's best practices to provide cost-effective solutions and service excellence
