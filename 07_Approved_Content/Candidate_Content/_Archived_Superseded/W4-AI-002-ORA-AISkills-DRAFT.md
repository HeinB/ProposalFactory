# Oracle AI-Based Skills Intelligence

**Capability Statement | APPSolve | Oracle Business Unit**

---

## Section 1: Statement of Capability

APPSolve configures and implements Oracle AI-Based Skills Intelligence within Oracle Fusion HCM — Oracle's embedded artificial intelligence framework for dynamic skills profile management, AI-powered talent matching, and data-driven career development.

Oracle AI Skills is a standard capability within the Oracle HCM Cloud platform, enabling organisations to build and maintain dynamic, continuously updated skills taxonomies for every employee. APPSolve delivers AI Skills as an integrated component of Oracle Fusion HCM Talent Management implementations.

> **Confirmed implementation:** Oracle AI-Based Skills was confirmed as a delivered component of the Hollywood Bets Oracle Fusion HCM Phase 4 (Talent Management) implementation (BU Lead decision 2026-06-14).

| Capability Area | Coverage | APPSolve Positioning |
|---|---|---|
| **Dynamic Skills Profiles** | AI-generated and maintained employee skills profiles — inferred from job history, performance, learning, and role data | **Confirmed delivery capability** — AI skills profile configuration within Oracle HCM |
| **Skills Gap Analysis** | Automated identification of skills gaps against role requirements and career targets | **Confirmed delivery capability** — organisation-wide skills gap visibility |
| **AI-Powered Job Recommendations** | Oracle AI suggests internal opportunities, gigs, and career pathways based on skills profiles | **Confirmed delivery capability** — talent marketplace recommendations within HCM |
| **Skills Taxonomy Management** | Configuration of skills libraries, proficiency levels, and role-to-skill mappings | **Confirmed delivery capability** — APPSolve configures Oracle's skills taxonomy to client requirements |
| **Manager Talent Intelligence** | Aggregated team skills view for managers — identify team-level gaps, succession risks, and development priorities | **Confirmed delivery capability** — manager analytics within Oracle Talent framework |
| **Learning Integration** | AI Skills aligned with Oracle Learning Cloud (W3S1-004) for personalised learning pathway recommendations | **Confirmed delivery capability** — native integration within Oracle HCM suite |

## Section 2: Product Architecture

### 2.1 Oracle AI Skills Integration within Fusion HCM

Oracle AI Skills operates as a native intelligence layer within Oracle Fusion HCM. It requires no separate middleware or external AI platform.

```
Oracle Fusion HCM Data Sources
  ├── Global HR — job histories, roles, assignments (W3S1-001)
  ├── Performance Management — goals, ratings, competencies (W3S1-002)
  ├── Learning Cloud — completed courses, certificates (W3S1-004)
  ├── Recruiting — job profiles, requisition requirements (W3S1-003)
  └── Talent Review — potential/performance calibration (W3S1-002)
         │
         ▼
Oracle AI Skills Engine (embedded)
  ├── Skills inference — derive skills from activity data
  ├── Skills taxonomy — Oracle global skills library + client configuration
  ├── Gap analysis — role vs. actual skills comparison
  └── Recommendation engine — career paths, opportunities, learning
         │
         ▼
Oracle HCM Employee Experience
  ├── Employee: My Skills profile, career recommendations, learning suggestions
  ├── Manager: Team skills dashboard, talent risk identification
  └── HR: Organisation-wide skills analytics, workforce planning inputs
```

### 2.2 Relationship to Oracle Talent Management Suite (W3S1-002)

Oracle AI Skills is a sub-module within the Oracle Talent Management Suite. W3S1-002 covers the broader Talent Management Suite (Performance, Goals, Talent Review, Career Development). W4-AI-002 specifically addresses the AI intelligence layer within that suite — extractable as a standalone capability where AI-specific tender requirements arise.

## Section 5: Approved and Prohibited Wording

### Approved Wording

> "APPSolve configures and implements Oracle AI-Based Skills Intelligence within Oracle Fusion HCM, enabling organisations to build dynamic employee skills profiles, identify skills gaps, and provide AI-powered career development recommendations."

> "APPSolve has delivered Oracle AI Skills as an integrated component of Oracle Fusion HCM Talent Management implementations, providing organisations with organisation-wide skills visibility and AI-driven talent development capabilities."

> "Oracle's embedded AI skills intelligence, configured by APPSolve, enables organisations to move from static, manually maintained skills records to dynamically inferred, AI-maintained skills profiles that improve talent decisions, succession planning, and learning investment."

### Prohibited Wording

- Do NOT reference SAA as a client, example, or implementation (Rule 21.1)
- Do NOT reference CCBA as a client or example (HIST-014)
- Do NOT reference DFA as a client or example (Rule 21.4)
- Do NOT describe Oracle AI Skills as a standalone AI product separate from Oracle HCM — it is embedded within the Oracle HCM Cloud platform
- Do NOT conflate Oracle AI Skills with Oracle Digital Assistant (ODA) — AI Skills manages skills intelligence; ODA is a conversational chatbot
- Do NOT conflate Oracle AI Skills with Oracle Grow — Grow is a talent marketplace/gig platform; AI Skills is the intelligence layer that powers recommendations
- Do NOT describe APPSolve as an AI developer or AI platform vendor — APPSolve configures Oracle's embedded AI capabilities within Oracle HCM implementations
