# Reference Gap Analysis

**Assessment date:** 2026-06-15 | **Active references assessed:** 16

---

## Executive Summary

APPSolve has a well-stocked Oracle reference corpus with 11 active Oracle-BU letters, including an Oracle Corporation partner endorsement. The Acumatica corpus has 5 active letters and is adequate for standard distribution tenders. Key gaps are: no standalone BeBanking reference, no Oracle HCM Fusion full-suite reference, no KPMG letter (despite templates sent), and no readable Harmony Gold letter (which would make three mining references). Scanned/unreadable PDFs represent 11 letters of potential untapped value.
