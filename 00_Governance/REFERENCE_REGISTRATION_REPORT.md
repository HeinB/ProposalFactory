Now that letters are registered, the following pre-tender checks apply when using registered clients:

1. **Check the NAMED_REFERENCE_MATRIX.md** — confirm the client is listed as "letter registered" (✅) before citing.

3. **Check scope** — Mr Price: Learning Cloud only; Cape Union Mart: Finance and Procurement only; DSSSA/FuelU2/Maxiflex: confirm current support relationship active.
4. **Confirm BEE certificate currency** — expires 2026-07-31; do not cite after expiry without renewal.
