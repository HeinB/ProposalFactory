# Tender Question Taxonomy
**Work Package:** WP5 — Tender Response Factory | **Version:** 1.0 | **Created:** 2026-06-14
**Purpose:** Maps common RFP/tender question patterns to approved KB assets and pre-composed response components. Use this to rapidly identify which assets to use when answering a specific tender question.

**How to use:**
1. Identify the question category from the tender
2. Find the matching question pattern in this taxonomy
3. Note the primary and supplementary assets listed
4. Apply governance checks (named reference restrictions, section exclusions)
5. Assemble response using ANSWER_ASSEMBLY_FRAMEWORK.md

**Readiness codes:** ✅ Ready (assets exist; pull and adapt) | ⚠️ Partial (assets exist but need augmentation) | ❌ Gap (no approved content; must author or escalate)
